# 排名轮播表

排名轮播表同轮播表类似，也可以选择单条轮播或整页轮播。

<dv-scroll-ranking-board :config="config" style="width:500px;height:300px" />

# config属性

属性 说明 类型 可选值 默认值
data 表数据 Array<Object> data属性 []
rowNum 表行数 Number --- 5
waitTime 轮播时间间隔(ms) Number --- 2000
carousel 轮播方式 String 'single'|'page' 'single'
unit 数值单位 String --- ''
